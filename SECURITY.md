# Security Policy

## Supported Versions

We actively support and patch security issues on the following versions:

| Version | Supported |
| :--- | :--- |
| 1.2.x | Yes (Active) |
| 1.1.x | Yes (Critical patches only) |
| < 1.1.0 | No |

## Reporting a Vulnerability

If you discover a security vulnerability within CareerOS AI, please do **not** open a public issue. Instead, report it privately to our security team.

1.  **Email:** Send an email to `security@careeros.ai`.
2.  **Details:** Include details on the vulnerability, steps to reproduce, and any proof of concept scripts or configuration settings.
3.  **Timeline:** We will acknowledge receipt of your report within 48 hours and work with you to patch and release a fix within 30 days.
