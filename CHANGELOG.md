# Changelog

All notable changes to **CareerOS AI** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.2.0] - 2026-07-16

### Added
- **Kanban Job Application Board:** Interactive board to track candidate job applications from wishlist to offer.
- **Milestone Career Roadmaps:** Automated learning paths with current/target skills checklists.
- **Cover Letter Generator:** Custom letter builder based on specific resume versions.
- **LinkedIn Profile Optimizer:** Tailored recommendations to sync resume strengths with LinkedIn search parameters.
- **Interview Preparation Simulator:** Audio/text dynamic questions with immediate score grades and improvement feedback.
- **Recruiter Portal:** Specialized panel for screening candidate profiles, search filters, and resumes.
- **EmptyState & ConfirmDialog:** Reusable shared UI elements to standardize empty listings and confirmation popups.
- **Accessibility Enhancements:** Lang tags, keyboard tab index navigation helper, skip-to-content anchors, and prefers-reduced-motion media controls.
- **One-Click Seeding Script:** Automated candidate and recruiter seeding workspace data script.

## [1.1.0] - 2026-06-10

### Added
- **Evidence Mode Audits:** Deep-linking and credibility engine to audit resume assertions.
- **Optimistic Concurrency Control (OCC):** Prevent dirty writes and overwrite errors on resumes.
- **Redis Rate Limiter:** Advanced bucket-based security on authentication endpoints.

## [1.0.0] - 2026-05-01

### Added
- Initial release containing core resume builder, live rendering PDF compile engine, ATS analyzer, and JD match scorer.
